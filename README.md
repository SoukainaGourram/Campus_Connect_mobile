# 🎓 Campus Connect

Campus Connect est une application mobile Android qui centralise les services universitaires pour les étudiants, professeurs et administration.

L’objectif est de faciliter la communication, l’accès aux informations et le partage des ressources dans une seule plateforme.

---

## 🚀 Fonctionnalités

- 📣 Annonces en temps réel
- 📅 Emploi du temps
- 💬 Chat privé et global
- 👥 Groupes de cours
- 📄 Partage de documents
- 👤 Gestion du profil
- 🔔 Notifications push

---

## 👥 Utilisateurs

### 👨‍🎓 Étudiants
- Consulter annonces
- Voir emploi du temps
- Rejoindre groupes
- Télécharger documents
- Contacter professeurs

### 👨‍🏫 Professeurs
- Créer groupes
- Partager cours et PDF
- Publier annonces
- Répondre aux étudiants

### 👨‍💼 Administration
- Gérer utilisateurs
- Publier annonces globales
- Superviser la plateforme

- Firebase Storage

---

## 📌 Roadmap

### MVP1
Application Android complète avec Firebase :
authentification, annonces, chat, groupes, documents.

### MVP2
Backend personnalisé avec FastAPI + PostgreSQL + Docker.

### MVP3
Google Maps + GPS pour localiser les salles.

### MVP4
Caméra + commandes vocales + chatbot IA.

### MVP5
Assistant personnel intelligent pour l’étudiant.

---

## 🎯 Objectif

Créer une plateforme universitaire moderne, simple et intelligente.
